# WAM-Weighted-Average-Mark-GPA-Calculator-Manager
A simple beginner project for personal academic score management and GPA calculation. (PS. The GPA calculation is only for 4.0-point GPA scale with custom percentage-to-grade-point mapping: 87-90: 3.7    84-87: 3.3   80-84: 3.0...and so on. ) (PPS.  The WAM is universally. )
